# 年表

**戦後〜2000年代初頭：国際経済・通貨体制の主要事件整理**

| **年** | **事件・枠組み** | **アメリカ・世界の動き** | **日本への影響・位置づけ** | **語呂合わせ** |
| --- | --- | --- | --- | --- |
| 1944 | ブレトン・ウッズ協定 | 米ドルを基軸通貨とし、金＝ドル固定（1oz=35ドル）。IMF・世銀創設。 | 円は固定相場（1ドル=360円）。戦後復興・輸出主導成長の土台。 | 行く良し(1944)ブレトンウッズ |
| 1947 | GATT成立 | 自由貿易推進（関税引下げ）。多国間貿易体制の始まり。 | 輸出立国として恩恵。後の高度成長の外的条件。 | 行くよな(1947)ガット |
| 1950–53 | 朝鮮戦争 | 冷戦構造の固定化。米国は軍需拡大。 | 特需により急速回復（「特需景気」）。 |  |
| 1955-1973 | 高度経済成長期 | 冷戦下の国際経済安定。自由貿易体制の恩恵。 | 輸出主導でGDP急成長。重化学工業化。国民生活向上。 | 行くGO(1955)高度成長 |
| 1960年代 | ドル過剰（ドルの垂れ流し） | ベトナム戦争・福祉拡大で財政赤字。金準備不足が顕在化。 | 円高圧力の芽生え。固定相場の維持が困難に。 |  |
| 1971 | ニクソン・ショック | 金とドルの交換停止。ブレトン・ウッズ体制崩壊。 | 360円固定終了。為替不安定化→輸出構造転換を迫られる。 | 行くない(1971)ニクソン |
| 1971 | スミソニアン協定 | 主要国がドル切下げ・固定相場再調整で合意（1ドル=308円）。 | 円切上げで輸出競争力低下の懸念。しかし暫定措置に過ぎず。 |  |
| 1973 | 変動相場制へ移行 | 各国が為替を市場に委ねる。 | 円高・円安リスクを初めて本格的に背負う。 | 行くなみ(1973)変動相場 |
| 1973 | 第一次オイルショック | OPECが原油価格引上げ。スタグフレーション発生。 | 資源輸入国として大打撃。省エネ・重厚長大型から転換。 |  |
| 1976 | キングストン合意（ジャマイカ合意） | 変動相場制を正式に承認。金の公式価格廃止。IMF体制の再編。 | 変動相場制が法的に確定。為替リスク管理が恒久的課題に。 |  |
| 1979 | 第二次オイルショック | イラン革命。インフレ再燃。 | 省エネ技術の競争力が日本優位に。 | 行くなく(1979)第二次オイル |
| 1985 | プラザ合意 | 米国主導でドル安誘導（対円・対マルク）。 | 急激な円高（輸出不振）→内需拡大策→バブルの温床。 | 行くわGO(1985)プラザ |
| 1987 | ルーブル合意 | 行き過ぎたドル安を是正。 | 金融緩和継続→資産価格高騰。 |  |
| 1991 | 冷戦終結 | 市場経済・自由貿易が「唯一の正解」に。 | 日本はバブル崩壊直後で停滞開始。 |  |
| 1993 | EU発足（マーストリヒト条約） | 欧州統合の深化。 | 日本企業は欧州市場戦略の再構築を迫られる。 |  |
| 1994 | ウルグアイ・ラウンド合意 | 農業・サービス・知財まで自由化対象拡大。 | 農業保護問題が政治争点化。 |  |
| 1995 | WTO設立 | GATTを引継ぎ、紛争処理を強化。 | 自由貿易体制の「ルール遵守国」として対応。 | 行くGO(1995)WTO |
| 1997 | アジア通貨危機 | 短期資本移動の危険性が顕在化。 | 日本の金融不安（山一破綻等）と連動。 |  |
| 2001 | 中国のWTO加盟 | グローバル化が新段階へ。 | 製造業の海外移転・空洞化が加速。 |  |

# **覚えるための「骨格（重要）」**

無秩序に覚えるな。3本の流れに分解しろ。

**① 通貨体制の流れ（最重要）**

- 固定相場制（ドル基軸）
→ ニクソンショック
→ 変動相場制
→ 円高問題（プラザ合意）

**② 資源・供給ショック**

- オイルショック
→ スタグフレーション
→ 日本：省エネ・技術立国化

**③ 貿易ルールの進化**

- GATT
→ ウルグアイ・ラウンド
→ WTO
→ 中国参加で競争激化

# **正直な指摘（ advisor モード ）**

- 今の君の理解は「単語暗記寄り」だ。
**「なぜ起きたか」「何を壊し、何を生んだか」**を言語化できていない。
- 共テ85超えを狙うなら
「アメリカの都合 → 世界制度変更 → 日本の構造変化」
この三段論法を即座に再生できる必要がある。

# **次にやるべき優先順位**

1. ニクソンショック〜プラザ合意を一本のストーリーで説明できるようにする
2. オイルショックが「物価」と「産業構造」に与えた影響を整理
3. WTO＝自由貿易の完成形ではなく「摩擦管理装置」だと理解する

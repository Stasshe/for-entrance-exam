| 要素    | 直列（電流共通）                                                                | 並列（電圧共通）                                                                |
| ----- | ----------------------------------------------------------------------- | ----------------------------------------------------------------------- |
| **R** | $V_R = I_0R\cos\omega t$                                                | $I_R = \dfrac{V_0}{R}\cos\omega t$                                      |
| **L** | $V_L = I_0\omega L\cos\left(\omega t + \dfrac{\pi}{2}\right)$           | $I_L = \dfrac{V_0}{\omega L}\cos\left(\omega t - \dfrac{\pi}{2}\right)$ |
| **C** | $V_C = \dfrac{I_0}{\omega C}\cos\left(\omega t - \dfrac{\pi}{2}\right)$ | $I_C = \omega C V_0\cos\left(\omega t + \dfrac{\pi}{2}\right)$          |

| 要素    | 直列：$Z$                 | 並列：$Y=1/Z$             |
| ----- | ---------------------- | ---------------------- |
| **R** | $R$                    | $\dfrac{1}{R}$         |
| **L** | $j\omega L$            | $-\dfrac{j}{\omega L}$ |
| **C** | $-\dfrac{j}{\omega C}$ | $j\omega C$            |
